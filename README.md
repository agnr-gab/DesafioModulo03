## **DESAFIO MÓDULO 3**

Faça um sistema responsável por registros de vendas de uma organização!

**Toda venda possui:**

 *1. um cliente,   
 2. vendedor responsável,  
 3.  valor a ser pago,  
 4. data de registro*
